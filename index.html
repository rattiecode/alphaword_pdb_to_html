AlphaWord PDB Encoder/Decoder

This window can convert PDB files to HTML. Right now, only bold and italic formatting is retained. This project is a work-in-progress! In the future, we plan to add functionality to convert HTML back to PDB.

Upload a .PDB file, and it will be converted to HTML and appear in the text box below.

File conversion is not processed by a server; it is handled within your browser window. Your text stays 100% in your browser and is never sent anywhere.

PDB → HTML

Select your PDB file using the button. Once it's selected, it will appear as formatted HTML in the box below and can be copied and used elsewhere.

HTML → PDB

To convert HTML to a PDB file, paste your formatted text into the text box at the bottom. Then enter a name for your file and click the Encode button. Another button will appear that will allow you to download your PDB file!
